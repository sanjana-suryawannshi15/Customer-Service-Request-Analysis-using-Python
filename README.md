# Customer-Service-Request-Analysis-using-Python
This project analyzes customer service request (311) data using Python to identify complaint patterns, major issues in different cities, and response times. It involves data cleaning, visualization, and basic statistical analysis to improve service efficiency and support better decision-making.

In this project, I worked on:
Cleaning and preparing raw data (handling missing values, correcting dates)
Performing data analysis to find trends and patterns
Identifying top complaint types across different cities
Visualizing data using graphs like bar charts, scatter plots, and heatmaps
Calculating response time to evaluate service efficiency
Applying statistical tests (Kruskal-Wallis test) to check differences in complaint resolution time

The goal of this project is to gain insights that can help improve customer service and decision-making.
